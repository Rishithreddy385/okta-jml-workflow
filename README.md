# Okta JML Workflow Project

End-to-end Joiner-Mover-Leaver workflow built on Okta, 
demonstrating identity lifecycle management for an enterprise environment.

## Scope
- SCIM provisioning from a mock HR source
- Group-based role assignment
- Access certification cycle
- Separation of Duties (SOD) policy
- Offboarding automation
